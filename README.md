# BiParking - Binusmaya Integrated Smart Parking Widget

An interface design project developed by Group 13 for COMP6800001 - Human and Computer Interaction.

## Design Philosophy: Less is More (Aesthetic & Minimalist Design)
Unlike standalone applications that require complex multi-screen navigation, BiParking is designed as a **Native Integrated Widget** inside the existing Binusmaya app. 

According to Human-Computer Interaction (HCI) principles for in-vehicle technology, user interfaces for drivers MUST minimize visual distraction and cognitive load. Therefore, our system strictly adheres to a **3-Screen Streamlined User Journey**:
1. **Dasbor Utama (Binusmaya Integration):** Instantly captures student attention with real-time occupancy metrics (Donut Chart) without requiring extra clicks.
2. **Detail Dasbor "Binus Parking":** Provides comprehensive matrix grid layout mapping (Color-coded: Blue for Available, Grey for Occupied) for immediate decision-making.
3. **Auto-Assign Notification Confirmation:** Displays automated slot allocation dynamically upon toll-gate scanning to ensure a seamless "first-come, first-served" user experience.

Creating extraneous, dozens of non-functional pages would directly violate the core HCI thesis of maximizing usability and speed for a user who is actively operating a motor vehicle.
